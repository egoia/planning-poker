Detailed Description

Contient les méthodes nécessaires à la logique du jeu.

Cette classe est responsable de la gestion des fonctionnalités, de la sauvegarde et du processus de jeu, en utilisant différents modes de jeu

int AppManager.joue_tour ( Card[]  cartes)
inline

Joue un tour en évaluant les cartes soumises par les joueurs.

Parameters
cartesTableau des cartes soumises par les joueurs pour ce tour
Returns
: Un code indiquant le résultat du tour :
  • '1' : Le tour s'est bien passé
  • '0' : Aucune carte validée
  • '-1' : Résultat joker
  • '-2' : Résultat "café"
  • '-3' : Aucune fonctionnalité restante à estimer

La fonction utilise une règle stricte pour le premier tour, puis applique la règle de validation sélectionnée pour les tours suivants
