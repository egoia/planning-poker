Detailed Description

Permet de gérer l'affichage lors du débat.

Cette classe est un composant qui s'occupe d'afficher les cartes extrêmes ainsi que les noms des joueurs qui ont voté avec ces cartes. Ce composant est activé uniquement lorsque le canva débat est affiché
